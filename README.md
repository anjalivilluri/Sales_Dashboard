📊 Sales Data Analysis Dashboard

Overview

Sales Data Analysis Dashboard is a responsive web application developed using HTML, CSS, JavaScript, and Chart.js.

The dashboard allows users to upload sales data in CSV format and visualize business insights through interactive charts, KPI cards, and searchable sales records.

---

Features

KPI Metrics

- Total Sales
- Total Orders
- Total Customers
- Average Sales

Data Visualization

- Monthly Sales Trend Chart
- Region Wise Sales Distribution Chart

Dashboard Features

- CSV File Upload
- Animated KPI Counters
- Search Sales Records
- Responsive Design
- Dark/Light Theme Toggle
- PDF Report Export

User Interface

- Modern Glassmorphism Design
- Mobile Responsive Layout

How to Run the Project

Step 1

Download or Clone the Repository.

Step 2

Open the project folder in Visual Studio Code.

Step 3

Install the Live Server Extension.

Step 4

Right-click on index.html.

Step 5

Click "Open with Live Server".

Step 6

Upload the sample sales.csv file.

Step 7

View the dashboard analytics.
